Student Record Management

Internship Information

  • Category: Native App Development
  • Github URL: Click Here

This application will be used for storage and display of student's personal data. It is used for storing and displaying student's marks and attendance. The application is developed using Flutter Technology.

Significance

  • The user can view personal data including attendance and marks

Implementation Details

  • This project is mainly used to demonstrate the foll. widgets:
    • Card
    • DataColumn
    • List View
    • DataRow
    • DataTable
    • DataCell
  • JSON format is used for storing the data
  • Stateful widgets are used for rendering UI components
  • Named Routes are used for routing purposes
