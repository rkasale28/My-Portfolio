This application is designed for users to provide information regarding college’s annual technical fest – Abhiyantriki. The content inspiration is taken from the official website of the event.

Significance

  • The application is designed such that our users – both from KJSCE and outside can easily navigate and use it
  • The app has its unique features apart from those on the website. Startup animation, sign-in support, live notifications, are amongst many that the app contains
  • The application allows user to sign into the experience and get information regarding all the events and competitions.
  • Registrations and contact details has been integrated within the application

Implementation Details

  • The application is built using Android Studio and different Google APIs to improve connectivity and productivity
  • For sign-in backend, Firebase is used
