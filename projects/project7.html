Covid Data Visualization

Internship Information

  • Category: Data Science, Web Development
  • Github URL: Click Here

This web application is made to help users visualize the Covid Data of India. This application was developed using Streamlit.

Significance

  • The users can view the data by Country Level, State Level or District Level.
  • The users can visualize the data by line graphs, bar graphs and pie charts.
  • The users can view:
    • On Country Level:
      • Daily Updates
      • State-wise breakdown
      • Cumulative Updates
      • Comparison between states
      • Breakdown
    • On State Level:
      • Daily Updates
      • District-wise breakdown
      • Cumulative Updates
      • Comparison between districts
      • Breakdown
    • On District Level:
      • Breakdown

    Please Note: Here, breakdown indicates breakdown of data into active, deceased, and recovered, whereas state-wise breakdown and district-wise breakdown indicates breakdown of data according to state and district, respectively.

Implementation Details

  • The data is sourced from:
    • https://api.covid19india.org/states_daily.json
    • https://api.covid19india.org/state_district_wise.json
    • https://api.covid19india.org/state_test_data.json
  • JSON data is converted into dataframe and stored in 3 csv files namely
    • district_level.csv
    • state_level_daily.csv
    • state_level_tested_daily.csv
  • The data is classified according to district, state and country
  • This web application is designed using Streamlit
